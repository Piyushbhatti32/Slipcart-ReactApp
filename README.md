# Slipcart
A simple e-commerce React application for buying and selling products.

## Clone
To clone the repository, run the following command in your terminal:
```
git clone https://github.com/piyushbhatti32/Slipcart-ReactApp.git
```

## Installation
To install the dependencies, run the following command in the project directory:
```
npm install
```
## Usage
To run the application, run the following command in the project directory:
```
npm start
```
